## 8. PRICING CALCULATOR LOGIC (Suggested Order of Operations)

```
1. SELECT base: Package (flat_rate) OR Service (per_hour × hours)
2. ADD any add-ons (each has its own price_type: flat_fee / per_event / per_artist / per_sq_ft / per_effect)
3. APPLY surcharges if applicable:
     - Exclusivity/NDA: multiply subtotal × 1.30 to 1.40
     - Seasonal Peak: multiply subtotal × 1.10 to 1.20
4. APPLY discounts if applicable:
     - Venue Owner repeat: subtract 10% from total
     - Radio Station ongoing: subtract 15% from total
5. CALCULATE deposit: total × 0.50
6. ADD event insurance if needed: $300–$400
7. DISPLAY: Subtotal → Surcharges → Discounts → Total → Deposit Due
```
